Projects

Below is a list of some of my web projects. For a full list of things I've worked on check out my GitHub profile.

  • base64
    Base 64 encoding/decoding tool. Simply encodes text into a base 64 string and decodes base 64 into a text string.
  • black
    Just a black screen. Useful for detecting stuck pixels, blocking distractions behind foreground windows, and reducing ambient light from always-on displays.
  • char
    Explore the unicode character set. Switch characters between different encoding schemes like hexadecimal, binary, CSS, JavaScript and HTML entities.
  • combinations
    Get each possible combination from all given inputs. When given a set of sets, this tool generates a table of all combinations of a single item from each set.
  • cursors
    Lists CSS cursor types. Hover over each for an example of what that cursor looks like in your browser. Test custom cursors, too.
  • embiggen
    Make text BIGGER. Resizes all text entered as you type to make it as big as possible on your screen but still fit.
  • global-composite-operation
    Demonstrates JS canvas composite operations. Lists all the operations and shows an example for each. Adjust the colors and see how the effect changes.
  • keyboard
    Simulates your keyboard on screen. See when keyup/keydown events are registered and the key codes associated with the key you just pressed.
  • opener
    Open a popup window to a location of your choosing with a given width and height. Useful if you want to take a screenshot just the right size.
  • sizer
    Check the inner and outer width and height of a window. Resize your window and see how the values change as you do.
  • text-cols
    Resize a textarea by columns. See how text-wrapping changes a body of text when adjusting the amount of space available to it.
  • uuids
    Generate multiple UUIDs for testing. Easily create hundreds of UUIDs and copy them to clipboard or just download them in JSON or CSV formats.
